# Palavrando

## Download das dependências
```
npm install
```

### Compila e habilita 'hot-reload' para desenvolvimento
```
npm run dev
```

### Compila e minifica o projeto para implantação
```
npm run build
```

### Aplica verificações estáticas no código
```
npm run lint
```
